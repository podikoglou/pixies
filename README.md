# pixies

What happens if you give an agent access to
[OpenStreetMap](https://www.openstreetmap.org)? Well, this project.

Pixies is an agent which can answer queries and uses OpenStreetMap's services
to answer them and display results in a map widget.

Services it uses:

- [https://nominatim.org/](Nominatim)
- [https://wiki.openstreetmap.org/wiki/Overpass](Overpass)
